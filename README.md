# GreenSumer_Back

Greensumer 서비스는 주변에 위치한 제로웨이스트 샵을 추천해주고 가게 이벤트와 리뷰를 사용자에게 공유하여 많은 사람들이 제로웨이스트 샵을 방문할 수 있도록 하기 위해 만들어진 서비스입니다.

## 개발 환경

- Intellij IDEA Ultimate 2023.1.3
- Java 17
- Gradle 7.4.1
- Spring Boot 2.7.6

## 기술 스택

- Spring Web
- Spring Data JPA
- Spring Security
- Spring Data Redis
- Spring Mail
- Spring Validation
- Spring Configuration Processor
- Lombok
- MySQL Driver
- JJWT
- Jackson Datatype
- Base62
- QueryDSL 5.0.0
- Swagger UI
- AssertJ
- jsr310

## 데모 페이지

현재 서비스는 Cloudtype에 배포된 상태입니다.

> https://web-greensumer-front-3xcah2glbj4mak4.gksl2.cloudtype.app/
